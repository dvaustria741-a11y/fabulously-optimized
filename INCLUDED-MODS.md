# SwiftMobile — Included Mods

> This file lists every mod bundled in SwiftMobile.
> Mods marked **[Mobile+]** are additions on top of the Fabulously Optimized base.

## Performance — Rendering

| Mod | Description |
|---|---|
| Sodium | Overhauled renderer — the foundation of any performance pack |
| Sodium Extra | Extra Sodium settings (particles, animations, etc.) |
| Reese's Sodium Options | Better Sodium options UI |
| ImmediatelyFast | Speed-up for immediate-mode rendering (UI, entities) |
| Enhanced Block Entities | Faster block-entity (chest, sign, banner) rendering |
| Entity Model Features | EMF — custom entity model support |
| Entity Texture Features | ETF — custom entity texture support |
| EntityCulling | Async path-tracing to skip off-screen entities |
| MoreCulling | Extra geometry culling passes |
| **[Mobile+] Exordium** | Renders HUD/GUI at lower FPS to free GPU for the world |
